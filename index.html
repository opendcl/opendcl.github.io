OpenDCL

OpenDCL is for AutoLISP programmers who want to replace AutoCAD’s limited DCL (Dialog Control Language) with a rich set of modern Windows controls and forms. It is free software under the GNU GPL, maintained by the OpenDCL Consortium.

What you get

  • OpenDCL Studio — visual dialog editor (installs runtime too)
  • OpenDCL Runtime — ARX/BRX (and related) modules for end-user deployment
  • Support across AutoCAD-compatible platforms (see release notes for versions)

This site is the GitHub Pages project home for downloads and help. Historic forum threads are archived at opendcl.github.io/forum-archive. The legacy live forum remains at www.opendcl.com/forum until freeze.
