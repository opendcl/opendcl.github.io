Имя функции dcl-AttachXref
Синтаксис   (dcl-AttachXref ФайлDwg [типа String] {ИмяБлока [типа String]} {ТочкаВставки [типа 3DPoint]} {Вращение [типа Float]})
Возвращаемый тип Boolean (T - успешно; иначе NIL)

Эта функция вставляет файл ФайлDwg как внешнюю ссылку в текущий чертеж. Если ТочкаВставки не указана, внешняя ссылка вставляется в начале координат. Если Вращение не указано, внешняя ссылка вставляется с углом поворота равным нулю. Если ИмяБлока не указано, чертеж файла будет использоваться в качестве имени внешней ссылки в текущем чертеже.

Эта функция может быть использована для подключения внешней ссылки в тех случаях, когда команда XREF не может быть использована (например от синхронного функции обработчика событий).
