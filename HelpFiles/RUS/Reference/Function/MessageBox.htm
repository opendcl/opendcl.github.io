Имя функции dcl-MessageBox
Синтаксис  

(dcl-MessageBox Текст [типа String] {Заголовок [типа String]} {ТипКнопок [типа Integer]} {ТипЗначка [типа Integer]} {ПоказатьПомощь [типа Boolean]} {LanguageId [as Integer]})
Возвращаемый тип Integer

Эта функция отображает стандартное окно сообщения Windows, с указанным текстом сообщения. Аргумент "Заголовок" задает заголовок окна сообщения, а остальные аргументы контролируют количество кнопок, их тип и значок, отображаемый рядом с сообщением. Окно сообщения по умолчанию отображаются только с кнопкой [OK] и без значка. Возвращаемое значение указывает, какая кнопка была нажата перед закрытием окна сообщения.
