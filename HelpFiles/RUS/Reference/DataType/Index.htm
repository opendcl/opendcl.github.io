OpenDCL поддерживает следующие типы данных, используемые в аргументах функций, и типы, возвращаемые функциями.

  • 2DPoint
  • 3DPoint
  • AxObject
  • Bitflag
  • Boolean
  • Color
  • Control
  • Date
  • Float
  • Handle
  • Image List
  • Integer
  • Long
  • OLEColor
  • PictureID
  • String
