Каждый загруженный проект OpenDCL идентифицируется по уникальному ключу. Ключом проектом является уникальная строка, которая назначается во время разработки и сохраняется в файле проекта. Можно переопределить сохраненный ключ проекта во время выполнения, указав новый ключ в методе Load. Документация OpenDCL использует обозначение <ПРОЕКТ> если ключ проекта должен быть передан в качестве аргумента.
