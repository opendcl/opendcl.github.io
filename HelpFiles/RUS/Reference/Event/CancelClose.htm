Имя функции c:<CONTROL-NAME>#OnCancelClose
Синтаксис   (defun c:<CONTROL-NAME>#OnCancelClose (Причина [типа Integer] /)
)
Возвращаемый тип Boolean
Подходит к File Dialog, Modal Dialog, Modeless Dialog

Данное событие срабатывает, когда пользователь пытается закрыть окно. Причина равна 1, если диалог закрывается потому что пользователь хочет отменить изменения; в противном случае он равен 0. Возвращенное значение T предотвращаетзакрытие диалог; в противном случае диалог будет закрыт.
