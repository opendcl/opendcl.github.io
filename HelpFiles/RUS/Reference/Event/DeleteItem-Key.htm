Имя функции c:<CONTROL-NAME>#OnDeleteItem
Синтаксис   (defun c:<CONTROL-NAME>#OnDeleteItem (Название [типа String] Ключ [типа Handle или String] /)
)
Возвращаемый тип Ignored
Подходит к Tree

Данное событие срабатывает, когда элемент был удален. "Название" является расширенным названием элемента. Ключ идентифицирует элемент; это строка, если элемент управления использует строковые ключи; в противном случае он является уникальным указателем, идентифицирующим элемент.
