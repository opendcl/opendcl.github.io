Имя функции c:<CONTROL-NAME>#OnDragnDropFromControl
Синтаксис   (defun c:<CONTROL-NAME>#OnDragnDropFromControl (ИмяПроекта [типа String] ИмяФормы [типа String] ИмяЭлемента [типа String] ТочкаПадения [типа Разные] /)
)
Возвращаемый тип Ignored
Подходит к Block List, Block View, DWG List, DWG Preview, Graphic Button, Hatch, Label, List Box, List View, Picture Box, Slide View, Text Box, Text Button, Tree

Данное событие срабатывает, когда пользователь завершает перетаскивание другого элемента управления OpenDCL на этот элемент. Аргументы ИмяПроекта, ИмяФормы и ИмяЭлемента идентифицируют элемент управления, из которого была инициирована операция перетаскивания. Аргумент ТочкаПадения является 2DPoint в большинстве случаев, за исключением следующих:

Элемент управления, получающий перетаскиваемый объект
 Аргумент ТочкаПадения
Tree Control с ключами элементов String (ключ элемента в точке падения)
Tree Control без ключей элементов Handle (указатель элемента в точке падения)
Все остальные элементы управления 2DPoint (координаты точки падения в клиентском элементе)
