Имя функции c:<CONTROL-NAME>#OnPaint
Синтаксис   (defun c:<CONTROL-NAME>#OnPaint (ИмеетФокус [типа Boolean] /)
)
Возвращаемый тип Ignored
Подходит к Picture Box, Slide View

Данное событие срабатывает, когда ОС Windows перерисовывает элемент. Если какая-либо временная графика была сделана на элементе, она должна быть перерисована. Это событие может быть вызвано часто, так что любой код в обработчике событий должен быть быстрым и оптимизированным.
