ToolTip Picture

Friendly Name ToolTip Picture
API Name ToolTipPicture
Type PictureID
Restrictions -1, or a valid picture ID
Get Function   (dcl-Control-GetToolTipPicture <CONTROL>)
Set Function   (dcl-Control-SetToolTipPicture <CONTROL> NewValue [as PictureID])
Applies To ActiveX Control, Angle Slider, Block List, Block View, Calendar, Check Box, Combo Box, DWG List, DWG Preview, Graphic Button, Grid, Hatch, Hyperlink, Image Combo Box, Label, List Box, List View, Option Button, Picture Box, Progress Bar, Slide View, Text Box, Text Button, Tree

The tooltip picture is displayed to the left of the main text when the tooltip is displayed.
