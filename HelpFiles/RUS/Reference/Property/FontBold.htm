Friendly Name Font Bold
API Name FontBold
Type Boolean
Restrictions  
Get Function   (dcl-Control-GetFontBold <CONTROL>)
Set Function   (dcl-Control-SetFontBold <CONTROL> NewValue [as Boolean])
Applies To Angle Slider, Block List, Calendar, Check Box, Combo Box, DWG List, Frame, Graphic Button, Grid, Hyperlink, Image Combo Box, Label, List Box, List View, Option Button, Option List, Picture Box, Tab Strip, Text Box, Text Button, Tree

If this property is set, the control's font is bold.
