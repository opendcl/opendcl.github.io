Friendly Name Hatch Scale
API Name HatchScale
Type Float
Restrictions  
Get Function   (dcl-Control-GetHatchScale <CONTROL>)
Set Function   (dcl-Control-SetHatchScale <CONTROL> NewValue [as Float])
Applies To Hatch

This property sets the scale factor used to display the hatch pattern. A scale of 1.0 draws the pattern at its nominal size.
