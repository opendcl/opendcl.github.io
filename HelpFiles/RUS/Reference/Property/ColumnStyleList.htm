Friendly Name Column Styles
API Name ColumnStyleList
Type List of Integers as (Column0Style Column1Style ...)
Restrictions Range is 0 - 43
Get Function   (dcl-Control-GetColumnStyleList <CONTROL>)
Set Function   (dcl-Control-SetColumnStyleList <CONTROL> NewValue [as List of Integers])
Applies To Grid

This property contains a list of default cell styles for each column. Cells in the column will use the default style unless a style override is set for the individual cell.
