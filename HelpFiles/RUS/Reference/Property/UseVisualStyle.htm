Friendly Name Use Visual Style
API Name UseVisualStyle
Type Boolean
Restrictions  
Get Function   (dcl-Control-GetUseVisualStyle <CONTROL>)
Set Function   (dcl-Control-SetUseVisualStyle <CONTROL> NewValue [as Boolean])
Applies To Angle Slider, Check Box, Combo Box, Frame, Image Combo Box, Option Button, Progress Bar, Scroll Bar, Spin Button, Slider, Tab Strip, Text Button

When this property is set, the control is drawn using Windows' visual styles (also known as "themes") when visual styles are enabled in versions of Windows that support them. Note that visual styles may override the control's background and foreground color settings.
