Support Forum

Friendly Name Width
API Name Width
Type Long
Restrictions Must be greater than zero
Get Function   (dcl-Control-GetWidth <CONTROL>)
Set Function   (dcl-Control-SetWidth <CONTROL> NewValue [as Long])
Applies To All Controls and Forms

This property determines the width of the control (in pixels). The value of this property is synchronized with the RightFromRight property. The Use Right From Right property controls the relationship of these properties.
