Friendly Name Title Bar
API Name TitleBar
Type Boolean
Restrictions  
Get Function   (dcl-Control-GetTitleBar <CONTROL>)
Set Function   (dcl-Control-SetTitleBar <CONTROL> NewValue [as Boolean])
Applies To Modal Dialog, Modeless Dialog

If this property is set, the form is displayed with a title bar; otherwise no title bar is displayed.
