Имя функции dcl-Form-Center
Синтаксис   (dcl-Form-Center <CONTROL> {NewWidth [типа Long]} {NewHeight [типа Long]})
Возвращаемый тип Boolean (T - успешно; иначе NIL)
Подходит к Control Bar, File Dialog, Modal Dialog, Modeless Dialog, Palette

This method centers the dialog on the display. If the optional NewWidth and NewHeight arguments are included, the form is resized before it is centered.
