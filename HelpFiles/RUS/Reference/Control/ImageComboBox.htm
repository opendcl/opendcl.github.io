Image Combo Box
Форум поддержки
Сообщить о баге

Выпадающий список с изображениями и/или текстовыми элементами, которые пользователь может выбрать. Также может иметь поле ввода, что пользователь мог ввести любой текст. Свойство Стиль определяет тип выпадающего списка.
