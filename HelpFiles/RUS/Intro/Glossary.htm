Элемент управления

В OpenDCL, элемент управления - это отдельный элемент пользовательского интерфейса. Обычно элементы OpenDCL аналогичны элементам Windows (таких как кнопки, текстовые поля и т.д.), но некоторые из них являются более сложными и состоят из несколькихэлементов Windows, работащих сообща. Обратите внимание, что формы также являются элементами управления OpenDCL.

Форма

Форма представляет собой шаблон, который используется для создания диалогового окна внутри AutoCAD во время выполнения. OpenDCL Studio используется для создания и интерактивного редактирования форм, размещения элементов управления на форме и настройки всех их свойств.
