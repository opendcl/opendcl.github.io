Для установки OpenDCL Studio нужно просто загрузить и запустить MSI файл желаемой языковой версии со страницы загрузки OpenDCL Studio на SourceForge. OpenDCL Studio делает возможным создавать формы OpenDCL без лицензии на AutoCAD, однако рекомендуется, чтобы все работы производились на компьютере с установленным AutoCAD чтобы формы и события AutoLISP кода могли быть протестированы в процессе разработки. OpenDCL Studio включает в себя исполняемый модуль OpenDCL для загрузки и отображенияформ OpenDCL внутри AutoCAD.

OpenDCL Studio предназначена для локальной установки, а не на сетевом сервере. Во время установки исполняемый модуль OpenDCL будет зарегистрирован для запроса его загрузки во всех поддерживаемых версиях AutoCAD, установленных ​​на целевом компьютере.

OpenDCL примеры

OpenDCL Studio включает в себя многочисленные примеры, демонстрирующие различные типы форм и элементы управления, а также примеры того, как они могут быть использованы. Чтобы увидеть образцы в действии, запустите AutoCAD и загрузите файл "Samples\@AllSamples.lsp" (устанавливается в подпапку "RUS" основной папки OpenDCL Studio). Файл можно перетащить из проводника Windows в окно AutoCAD чтобы быстро увидеть OpenDCL в действии. Обратите внимание, что не требуется добавлять никакие папки в пути доступа к вспомогательным файламAutoCAD, чтобы загружать и использовать эти примеры.
