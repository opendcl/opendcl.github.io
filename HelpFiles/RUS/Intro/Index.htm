OpenDCL представляет собой приложение с открытым исходным кодом для проргаммистов AutoLISP которые хочет заменить  ограниченный язык пользовательского интерфейсаDCL (Dialog Control Language) AutoCAD'а с богатым набором современных элементов пользовательского интерфейса Windows. Разработчики приложений AutoCAD и пользователи могут использовать OpenDCL Studio чтобы интерактивно разрабатывать современные и профессиональные элементы пользовательского интерфейса для использования их вкоде AutoLISP. OpenDCL Runtime компонент развертывается вместе с приложением, и обеспечивает для конечных пользователей необходимую основу выполнения приложения OpenDCL внутри AutoCAD.
