Alt Orientation
Support Forum
Report A Bug
Friendly Name Alt Orientation
API Name AltOrientation
Type Integer
Restrictions Range is 0 - 1
Get Function   (dcl-Control-GetAltOrientation <CONTROL>)
Set Function   (dcl-Control-SetAltOrientation <CONTROL> NewValue [as Integer])
Applies To Grid

This property determines the orientation of the alternating color if one is used.

Value Description
0 Horizontal orientation (alternating rows)
1 Vertical orientation (alternating columns)
