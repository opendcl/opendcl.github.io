Friendly Name ToolTip Line
API Name ToolTipLine
Type Boolean
Restrictions  
Get Function   (dcl-Control-GetToolTipLine <CONTROL>)
Set Function   (dcl-Control-SetToolTipLine <CONTROL> NewValue [as Boolean])
Applies To ActiveX Control, Angle Slider, Block List, Block View, Calendar, Check Box, Combo Box, DWG List, DWG Preview, Graphic Button, Grid, Hatch, Hyperlink, Image Combo Box, Label, List Box, List View, Option Button, Picture Box, Progress Bar, Slide View, Text Box, Text Button, Tree

If this property is set, a horizontal separator line is displayed between the tooltip title and main text.
