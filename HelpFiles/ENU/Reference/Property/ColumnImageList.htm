Friendly Name Column Images
API Name ColumnImageList
Type List of Integers as (Column0HeaderImageIndex Column1HeaderImageIndex ...)
Restrictions Image index values must be valid images from the control's image list, or -1 for no image
Get Function   (dcl-Control-GetColumnImageList <CONTROL>)
Set Function   (dcl-Control-SetColumnImageList <CONTROL> NewValue [as List of Integers])
Applies To Grid

This property contains a list of header images for each column. The header image is displayed in the column header along with the column header caption.
