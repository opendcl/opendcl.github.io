Allow Resizing
Support Forum
Report A Bug
Friendly Name Allow Resizing
API Name AllowResizing
Type Boolean
Restrictions  
Get Function   (dcl-Control-GetAllowResizing <CONTROL>)
Set Function   (dcl-Control-SetAllowResizing <CONTROL> NewValue [as Boolean])
Applies To Control Bar, File Dialog, Modal Dialog, Modeless Dialog, Options Tab, Palette

If this property is set, the form is displayed with a resizing border that allows the dialog to be resized by dragging the border.
