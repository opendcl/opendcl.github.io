Friendly Name Top
API Name Top
Type Long
Restrictions  
Get Function   (dcl-Control-GetTop <CONTROL>)
Set Function   (dcl-Control-SetTop <CONTROL> NewValue [as Long])
Applies To All Controls

This property determines the Y coordinate (in pixels) of the upper left corner of the control. The coordinate is relative to the upper left corner of the control's parent window (higher values are lower on the display). The value of this property is synchronized with the Top From Bottom and Height properties. The Use Top From Bottom property controls the relationship of these properties.
