Column Widths
Support Forum
Report A Bug
Friendly Name Column Widths
API Name ColumnWidthList
Type List of Longs as (Column0Width Column1Width ...)
Restrictions Must be zero or greater
Get Function   (dcl-Control-GetColumnWidthList <CONTROL>)
Set Function   (dcl-Control-SetColumnWidthList <CONTROL> NewValue [as List of Longs])
Applies To Grid

This property contains a list of column widths (in pixels) for each column.
