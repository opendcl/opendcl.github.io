Style (Graphic Button)
Support Forum
Report A Bug
Friendly Name Style
API Name GraphicButtonStyle
Type Integer
Restrictions Range is 0 - 6
Get Function   (dcl-Control-GetGraphicButtonStyle <CONTROL>)
Set Function   (dcl-Control-SetGraphicButtonStyle <CONTROL> NewValue [as Integer])
Applies To Graphic Button

This property determines the style of the graphic button.

Value Meaning
0 Raised border
1 Flat button
2 Pick entity button
3 Selection set button
4 Entity filter button
5 Borderless button
6 XP themed button
