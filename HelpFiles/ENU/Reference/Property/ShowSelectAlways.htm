Show Select Always
Support Forum
Report A Bug
Friendly Name Show Select Always
API Name ShowSelectAlways
Type Boolean
Restrictions  
Get Function   (dcl-Control-GetShowSelectAlways <CONTROL>)
Set Function   (dcl-Control-SetShowSelectAlways <CONTROL> NewValue [as Boolean])
Applies To List View, Tree

If this property is set, the selection highlight is always shown even when the control does not have input focus; otherwise no selection is shown unless the control has the input focus.
