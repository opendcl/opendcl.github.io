Friendly Name Border Style
API Name BorderStyle
Type Integer
Restrictions Range is 0 - 2
Get Function   (dcl-Control-GetBorderStyle <CONTROL>)
Set Function   (dcl-Control-SetBorderStyle <CONTROL> NewValue [as Integer])
Applies To Animation, Block List, Block View, DWG List, DWG Preview, Grid, Hatch, Html, Label, List Box, List View, Option List, Picture Box, Progress Bar, Rectangle, Slide View, Text Box, Tree

The border style controls the style of border displayed around the control window.

Style Description
0 None
1 Client edge
2 Static edge
