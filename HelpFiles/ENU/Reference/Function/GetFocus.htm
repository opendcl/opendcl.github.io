Function Name dcl-GetFocus
Syntax   (dcl-GetFocus)
Return Type List of (<PROJECT> [as Project] Dialog [as String] Control [as String])

This function returns the project, dialog, and control keys of the control with the current keyboard input focus. If the focus control is not an OpenDCL control, the function returns NIL.
