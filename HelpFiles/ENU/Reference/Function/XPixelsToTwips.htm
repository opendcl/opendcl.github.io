Function Name dcl-XPixelsToTwips
Syntax   (dcl-XPixelsToTwips Pixels [as Long])
Return Type Long

This function converts a size in pixels to "twips". A twip is a typographical unit of measure equal to 1/20 of a typographical "point" (a point is 1/72 of an inch), so this conversion is dependent on the display resolution.
