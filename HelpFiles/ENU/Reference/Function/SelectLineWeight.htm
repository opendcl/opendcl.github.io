Function Name dcl-SelectLineWeight
Syntax   (dcl-SelectLineWeight {DefaultLineWeight [as Integer]} {AllowMetaTypes [as Boolean]})
Return Type Integer

This function displays the AutoCAD line weight selection dialog and returns the selected line weight. If AllowMetaTypes is NIL, the ByLayer and ByBlock linetypes will not be selectable. Line weights are expressed as integer values that are either logical values or the actual line weight in millimeters multiplied by 100.
