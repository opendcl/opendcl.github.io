Function Name dcl-MessageBox
Syntax  

(dcl-MessageBox Message [as String] {Title [as String]} {ButtonStyle [as Integer]} {IconStyle [as Integer]} {ShowHelpButton [as Boolean]} {LanguageId [as Integer]})
Return Type Integer

This function displays a standard Windows message box with the message text specified in Message. The Title argument specifies the message box title, and the remaining arguments control the number and type of buttons, and the icon displayed beside the message. If an optional language ID is provided, and the corresponding language pack is installed, the message box button captions will use the specified language instead of the default language. The default message box displays only an [OK] button with no icon. The return value indicates which button was pressed to close the message box.
