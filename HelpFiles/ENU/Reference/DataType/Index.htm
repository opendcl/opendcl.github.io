OpenDCL supports the following data types used for function arguments and function return types.

  • 2DPoint
  • 3DPoint
  • AxObject
  • Bitflag
  • Boolean
  • Color
  • Control
  • Date
  • Float
  • Handle
  • Image List
  • Integer
  • Long
  • OLEColor
  • PictureID
  • String
