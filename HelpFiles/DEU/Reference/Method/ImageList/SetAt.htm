Funktionsname dcl-ImageList-SetAt
Syntax   (dcl-ImageList-SetAt entImageList [als Symbolliste] intImageIndex [als Integer] strFilename [als String] {intWidth [als Long] intHeight [als Long]})
Rückgabewert Integer (neuer oder gelöschter Bild-Index bei Erfolg, NIL wenn es fehlgeschlagen ist)

Diese Funktion ergänzt, ersetzt oder löscht ein Symbol an der angegebenen Stelle (Bild-Index). Das erste Element in der Symbolliste hat immer den Index 0. Übergeben Sie den Bild-Index -1, um das angegebene Bild am Ende zu ergänzen. Ist der Dateiname nil, wird das Bild an der angegebenen Stelle entfernt. Die optionalen Werte für Breite und Höhe sollten übergeben werden, wenn Sie das erste Bild einer leeren Symbolliste hinzufügen. Ist die Dimension der Symbolliste bereits definiert, wird das erste hinzugefügte Symbol automatisch auf diese Größe neuberechnet. Alle anschließend hinzugefügten Symbole werden automatisch auf die Größe des ersten Symbols skaliert. Übergeben Sie Werte für Breite und Höhe für ein Bild einer bereits gefüllten Symbolliste, so müssen die angegebenen Werte der Dimension der Symbolliste entsprechen, andernfalls schlägt die Funktion fehl.
