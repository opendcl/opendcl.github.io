Funktionsname dcl-ListView-GetItemData
Syntax   (dcl-ListView-GetItemData <CONTROL> intItem [als Long])
Rückgabewert Long
Anwendbar auf List View

Diese Methode gibt nutzerspezifische Daten zurück, die dem angegebenen Eintrag mit der Methode SetItemData zugewiesen wurden (z.B. den eindeutigen Schlüssel eines Datensatzes aus einer Datenbanktabelle). Die Methode gibt den Wert NIL zurück, wenn dem angegebenen Eintrag zuvor keine nutzerspezifische Daten zugeordnet wurden. In einer Liste im Stil Details ist das Argument intColumn obligatorisch anzugeben, in den anderen Stilen der Liste ist die Angabe der Spalte nicht notwendig.
