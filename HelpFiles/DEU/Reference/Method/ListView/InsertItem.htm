Funktionsname dcl-ListView-InsertItem
Syntax   (dcl-ListView-InsertItem <CONTROL> (intItem [als Long] {intItemImageIndex [als Integer]} strItemLabel [als String] {strColumn1Label [als String]} {strColumn2Label [als String]} ...))
Rückgabewert Boolean (T, wenn erfolgreich, ansonsten NIL)
Anwendbar auf List View

Diese Methode fügt einen neuen Eintrag in der angegebenen Zeile intItem ein. Das optionale Argument intImageIndex gibt ein Bild aus der Bildliste des Steuerelements an, das neben der Bezeichnung in der ersten Spalte der neuen Zeile in der Darstellung "Details" bzw. "Symbol" zu sehen ist. In der Darstellung "Details" stellt jede Zelle der neuen Zeile den entsprechenden Text aus der übergebenen Liste der Bezeichnungen dar. Wird für eine Zelle kein Text angegeben, bleibt diese leer. Der Rückgabewert dieser Methode ist der Index der neuen Zeile.
