Deutschsprachiges Forum
Funktionsname dcl-Grid-GetRowCells
Syntax   (dcl-Grid-GetRowCells <CONTROL> intRow [als Long])
Rückgabewert Liste von Strings in der Form (strCell0Label strCell1Label ...)
Anwendbar auf Grid

Diese Methode gibt eine Liste mit allen Texten der jeweiligen Zellen der angegebenen Zeile zurück.
