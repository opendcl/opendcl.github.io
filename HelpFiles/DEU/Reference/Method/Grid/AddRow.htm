Funktionsname dcl-Grid-AddRow
Syntax   (dcl-Grid-AddRow <CONTROL> {intColumn0ImageIndex [als Integer]} strColumn0Label [als String] {strColumn1Label [als String] strColumn2Label [als String]} ...)
Rückgabewert Long
Anwendbar auf Grid

Diese Methode fügt eine neue Zeile am Ende des Datenblatts hinzu. Mit intColumnImageIndex kann optional ein Bild aus der Bildliste des Steuerelements übergeben werden, das neben der Bezeichnung angezeigt wird (Vorgabe: kein Bild). Verwenden Sie die Methode SetCellImages, um in den Zellen der anderen Spalten Symbole nachträglich anzuzeigen.

Jede Zelle in der neuen Zeile wird zunächst mit dem vordefinierten Text (aus der Argumentenliste) angezeigt. Wird für eine Zelle kein Text angegeben, bleibt sie leer. Wollen Sie numerische Werte darstellen, müssen Sie diese zuvor in Texte konvertieren. Neue Zellen erben zunächst die Eigenschaften der Spaltendefinition, wozu der Zellenstil und das vordefinierte Bild gehören. Der Stil der Zelle oder Spalte regelt nur das Verhalten der Zelle, und was geschieht, wenn der Benutzer auf die Zelle klickt. Zum Beispiel: Haben Sie beispielsweise den Stil 33 - Linientyp für eine der Zellen gewählt, dann muss die übergebene Auswahl für den Linientyp in den Methoden AddString, AddRow, InsertRow und InsertString richtig gesetzt werden, d.h. in diesem Fall muss der Linientyp existieren. Möglicherweise ist sonst das dargestellte Ergebnis nicht korrekt.

Der Rückgabewert dieser Methode ist der Index der neuen Zeile, beginnend mit 0 für die erste Zeile.

Beachten Sie dringend auch die weiterführenden zellabhängigen Informationen der Funktion SetCellStyle.
