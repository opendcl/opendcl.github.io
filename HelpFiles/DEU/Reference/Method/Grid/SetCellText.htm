Grid-SetCellText
Anmerkung zur deutschen Übersetzung melden
Deutschsprachiges Forum
Funktionsname dcl-Grid-SetCellText
Syntax   (dcl-Grid-SetCellText <CONTROL> intRow [als Long] intColumn [als Long] strLabel [als String])
Rückgabewert Boolean (T, wenn erfolgreich, ansonsten NIL)
Anwendbar auf Grid

Diese Methode weist der angegebenen Zelle einen neuen Text zu. Wollen Sie numerische Werte darstellen, müssen Sie diese zuvor in Texte konvertieren.

Beachten Sie dringend auch die weiterführenden zellabhängigen Informationen der Funktion SetCellStyle.
