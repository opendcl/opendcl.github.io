Funktionsname dcl-Grid-AddColumns
Syntax   (dcl-Grid-AddColumns <CONTROL> lstNewColumns [als Liste von Listen in der Form ((strCaption [als String] {intJustification [als Integer]} {intColumnWidth [als Long]} {intColumnImageIndex [als Integer]}) ...)])
Rückgabewert Boolean (T, wenn erfolgreich, ansonsten NIL)
Anwendbar auf Grid

Diese Methode fügt neue Spalten zum Datenblatt hinzu. Die Bezeichnung der Spalte wird über das Argument strCaption angegeben, intJustification definiert, wie die Spaltenbezeichnung ausgerichtet ist (Vorgabe: linksbündig). Das Argument intColumnWidth bestimmt die Spaltenbreite. Lassen Sie das Argument intColumnWidth aus, wird die Spaltenbreite aus der Breite der Spaltenbezeichnung berechnet. Mit intColumnImageIndex kann optional ein Bild aus der Bildliste des Steuerelements übergeben werden, das neben der Bezeichnung angezeigt wird (Vorgabe: kein Bild). Bitte beachten Sie, dass aufgrund von Einschränkungen im Windows-Grid-Steuerelement die Spalte 0 immer links ausgerichtet ist und dies kann nicht geändert werden.

Ausrichtung Stil
0 linksbündig
1 zentriert
2 rechtsbündig
