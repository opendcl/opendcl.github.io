Funktionsname dcl-AxControl-GetAxObject
Syntax   (dcl-AxControl-GetAxObject <CONTROL>)
Rückgabewert AxObject
Anwendbar auf ActiveX Control

Diese Methode gibt das Handle des ActiveX-Steuerelements zurück. Das Handle kann erforderlich sein, wenn Sie COM- oder AxObject Methoden ausführen wollen.
Wenn Sie das Objekt nicht mehr benötigen, sollten Sie die Funktion dcl-AxObject-Close anwenden, um das Objekt aus dem Speicher zu entfernen.
