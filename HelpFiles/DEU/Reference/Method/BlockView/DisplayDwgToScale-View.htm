Funktionsname dcl-BlockView-DisplayDwgToScale
Syntax   (dcl-BlockView-DisplayDwgToScale <CONTROL> strDwgFileName [als String] {intView [als Integer]} {reaZoomFactor [als Float]})
Rückgabewert Boolean (T, wenn erfolgreich, ansonsten NIL)
Anwendbar auf Block View

Diese Methode stellt eine externe Zeichnungsdatei dar. Das Argument intView bestimmt die Blickrichtung, das Argument reaZoomFactor den genauen Zoomfaktor für die Ansicht. Dabei gibt reaZoomFactor gibt den Zoomfaktor in AutoCAD-Einheiten pro Pixel für die Ansicht an. Hat das Argument reaZoomFactor den Wert 0, wird der Block auf seine Zeichnungsgrenzen herausgezoomt. Das ist gleichzeitig die Vorgabe, wenn reaZoomFactor weggelassen wird.

Wert Vordefinierte Ansicht
0 Oben
1 Unten
2 Rechts
3 Links
4 Vorn
5 Hinten
6 ISO SW
7 ISO SO
8 ISO NW
9 ISO NO
