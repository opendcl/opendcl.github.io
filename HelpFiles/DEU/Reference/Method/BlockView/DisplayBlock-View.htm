Funktionsname dcl-BlockView-DisplayBlock
Syntax   (dcl-BlockView-DisplayBlock <CONTROL> strBlockName [als String] {intView [als Integer]} {reaZoomFactor [als Float]})
Rückgabewert Boolean (T, wenn erfolgreich, ansonsten NIL)
Anwendbar auf Block View

Diese Methode lädt einen Block der aktuellen Zeichnung oder einer mit der Methode PreLoadDwg im Hintergrund geladenen Zeichnungsdatei. Das Argument intView bestimmt die Blickrichtung, das Argument reaZoomFactor den Zoomfaktor für die Ansicht. Größere Werte zeigen den Block größer an als kleinere Werte. Der Faktor 1.0 bestimmt, dass der Block in der Ansicht auf die Zeichnungsgrenzen gezoomt wird. Das ist gleichzeitig die Vorgabe, wenn reaZoomFactor weggelassen wird.

Wert Vordefinierte Ansicht
0 Oben
1 Unten
2 Rechts
3 Links
4 Vorn
5 Hinten
6 ISO SW
7 ISO SO
8 ISO NW
9 ISO NO
