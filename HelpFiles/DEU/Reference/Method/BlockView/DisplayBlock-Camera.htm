Funktionsname dcl-BlockView-DisplayBlock
Syntax   (dcl-BlockView-DisplayBlock <CONTROL> strBlockName [als String] reaCameraX [als Float] reaCameraY [als Float] {reaCameraZ [als Float]} {reaZoomFactor [als Float]})
Rückgabewert Boolean (T, wenn erfolgreich, ansonsten NIL)
Anwendbar auf Block View

Diese Methode lädt einen Block der aktuellen Zeichnung oder einer mit der Methode PreLoadDwg im Hintergrund geladenen Zeichnungsdatei. Die Anzeige wird über eine Kameraposition in Abhängigkeit des Zeichnungsursprung des Blockes eingestellt. Wird der Zoomfaktor mit angegeben, wird die Anzeige in Abhängigkeit von den Grenzen des Blockes bzw. der Datei skaliert. Zum Beispiel: 1.0 = Zoom Grenzen (ist zugleich Voreinstellung, wenn dieses Argument ausgelassen wird), 0.5 = 1/2 stellt den Block nur halb so groß dar und 2.0 = zoomt sich in den Block und stellt ihn doppelt so groß dar.
