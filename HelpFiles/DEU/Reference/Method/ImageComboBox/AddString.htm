Funktionsname dcl-ImageComboBox-AddString
Syntax   (dcl-ImageComboBox-AddString <CONTROL> strNewItem [als String] intImageIndex [als Integer] {intSelectedImageIndex [als Integer]} {intIndent [als Integer]} {intIndex [als Integer]})
Rückgabewert Long
Anwendbar auf Kombinationsfeld mit Bildern

Diese Methode fügt eine neue Zeile in der Liste an oder in die Liste an der angegebenen Stelle ein. Die Argumente intImageIndex und intSelectedImageIndex sind Bilder, die angezeigt werden können, wenn ein Eintrag ausgewählt oder abgewählt wird. Mit dem Argument intIndent geben Sie den linken Einzug in Pixel an, wenn ein neuer Eintrag hinzukommt. Auf diesem Wege können Sie Baumstrukturen in Auswahllisten darstellen. Wird das Argument intIndex nicht angegeben, wird das Element hinten angehangen. Der Rückgabewert ist der Index des neuen Eintrags.
