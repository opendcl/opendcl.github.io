Funktionsname dcl-Animation-Play
Syntax   (dcl-Animation-Play <CONTROL> {intFirstFrame [als Long]} {intLastFrame [als Long]} {intRepeatCount [als Long]})
Rückgabewert Boolean (T, wenn erfolgreich, ansonsten NIL)
Anwendbar auf Animation

Diese Methode startet die Animation bzw. spiel das Video ab. Ohne Angabe von Frame-Indizes und Anzahl von Wiederholungen startet das Video im ersten Frame, und es wird automatisch unendlich oft wiederholt.
Bei einem Wert von -1 als End-Frame wird das Video bis zum letzten Bild abgespielt.
Übergeben Sie für die Anzahl an Wiederholungen eine -1, wird das Video automatisch unendlich wiederholt.

Argument Vorgabewert
intFirstFrame 0 (erstes Bild)
intLastFrame -1 (letztes Bild)
intRepeatCount -1 (unendlich oft wiederholen)
