Funktionsname dcl-AxObject-Put
Syntax   (dcl-AxObject-Put <AXOBJECT> strPropertyName [als String] uNewValue [variiert])
Rückgabewert Boolean (T, wenn erfolgreich, ansonsten NIL)
Anwendbar auf ActiveX Object

Diese Methode setzt Eigenschaftswerte für das ActiveX-Objekt. Der Datentyp des Eigenschaftswerts wird von der Eigenschaft des ActiveX-Objekts vorgegeben und kann daher unterschiedlich ausfallen. Deshalb müssen die Datentypen der Argumente zur Eigenschaft passen oder zumindest vom Interpreter in die richtigen Datentypen konvertiert werden können. Der Rückgabewert ist NIL, wenn die Zuwesing fehlgeschlagen ist. Lesen Sie hierzu auch dringend die Dokumentation zur API des ActiveX-Objekts.
