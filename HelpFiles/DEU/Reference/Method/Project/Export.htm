Funktionsname dcl-Project-Export
Syntax   (dcl-Project-Export <PROJECT> {strPassword [als String]})
Rückgabewert String

Diese Methode exportiert ein geladenes Projekt in Base-64 codierte binäre Projektdaten. Ist das Projekt passwortgeschützt, muss über das optionale Argument strPassword das passende Passwort übergeben werden, andernfalls wird die Funktion fehlschlagen.

Die codierten Daten werden von dieser Methode in einer Form zurückgegeben, die sehr einfach innerhalb des AutoLISP-Quellcodes eingebettet und später mit Hilfe der Methode Import geladen werden können. Das Einbetten eines OpenDCL-Projekts innerhalb des kompilierten AutoLISP-Quellcodes ist ein guter Weg, die Projektdateien vor dem Zugriff Dritter zu schützen. Außerdem können keine Probleme durch das Löschen externer Dateien entstehen.
