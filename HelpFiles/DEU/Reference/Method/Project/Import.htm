Funktionsname dcl-Project-Import
Syntax   (dcl-Project-Import strProjectData [als String oder als Liste von Strings] {strNewPassword [als String]} {strNewProjectKey [als String]})
Rückgabewert String

Diese Methode importiert ein Projekt aus Base-64 codierten Projektdaten. Die codierten Daten kann jede Base-64 codierte *.odcl-Datei (eine Projektdatei, die mit der Dateierweiterung *.lsp in OpenDCL Studio gespeichert wurde, ist ebenfalls ein Base-64 codiertes Format) oder die zurückgegebene Zeichenkette der Methode Export. Das optional übergebene Argument strNewPassword ersetzt ein etwaig vorhandenes Passwort. Das optional übergebene Argument strNewProjectKey ersetzt den Projektschlüssel, der in den Projektdaten gespeichert ist. War der Import erfolgreich, gibt die Methode den Projektschlüssel des neuen Projekts zurück, ansonsten NIL.
