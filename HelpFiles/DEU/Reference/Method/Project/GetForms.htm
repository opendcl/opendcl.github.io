Funktionsname dcl-Project-GetForms
Syntax   (dcl-Project-GetForms <PROJECT> {strPassword [als String]})
Rückgabewert Liste von Dialogen

Diese Methode gibt eine Liste aller in dem angegebenen Projekt definierten Dialoge zurück. Ist das Projekt durch ein Passwort geschützt, muss über das optionale Argument strPassword das passende Passwort übergeben werden, andernfalls wird die Funktion fehlschlagen.

Bitte beachten Sie, dass jeder Dialog ein verstecktes Steuerelement enthält, dass die Eigenschaften der Dialogbox selbst besitzt. Ein Dialog wird in AutoLISP über dieses versteckte Steuerelement abgebildet. Daher sind die Elemente dieser Liste, die von dieser methode zurückgegeben wird, einfache Objektzeiger der Steuerelemente.
