Funktionsname dcl-TabStrip-GetRowCount
Syntax   (dcl-TabStrip-GetRowCount <CONTROL>)
Rückgabewert Long
Anwendbar auf Tab Strip

Diese Methode gibt die aktuelle Anzahl der Zeilen aus, in der die Karteikartenreiter angezeigt werden.
