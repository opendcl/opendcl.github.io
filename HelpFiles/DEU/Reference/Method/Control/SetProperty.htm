Funktionsname dcl-Control-SetProperty
Syntax #1   (dcl-Control-SetProperty <CONTROL> strPropertyApiName [als String] uNewValue [variiert])
Syntax #2   (dcl-Control-SetProperty <CONTROL> (strPropertyApiName1 [als String] uNewValue1 [variiert])(strPropertyApiName2 [als String] uNewValue2 [variiert]) ...)
Rückgabewert Boolean (T, wenn erfolgreich, ansonsten NIL)
Anwendbar auf Alle Steuerelemente und Dialoge

Diese Methode setzt für ein Steuerelement mehrere Eigenschaften gleichzeitig. Die Methode gibt NIL zurück, wenn die Zuweisung bei einem oder mehreren Eigenschaften fehlgeschlagen ist.
