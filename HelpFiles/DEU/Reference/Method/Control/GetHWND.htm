Funktionsname dcl-Control-GetHWND
Syntax   (dcl-Control-GetHWND <CONTROL>)
Rückgabewert Handle
Anwendbar auf Alle Steuerelemente und Dialoge

Diese Methode gibt das Window handle (HWND) des Steuerelements bzw. Dialogs zurück.

Die hWnd-Eigenschaft gibt eine Zugriffsnummer für ein Formular oder Steuerelement zurück, d. h. ein eindeutiger ganzzahliger Wert, der von der Betriebssystemumgebung definiert wird und von einem Programm verwendet wird, um ein Objekt (zum Beispiel eine Form oder ein Steuerelement) zu kennzeichnen und zu einem Objekt zu wechseln. Meistens wird eine Zugriffsnummer auch als Handle bezeichnet.

Die hWnd-Eigenschaft wird im Zusammenhang mit Aufrufen der Windows-API verwendet. Viele Funktionen der Windows-Arbeitsumgebung benötigen die hWnd-Zugriffsnummer des aktiven Fensters als Argument.
