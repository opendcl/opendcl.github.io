Form-Hide

Deutschsprachiges Forum
Funktionsname dcl-Form-Hide
Syntax   (dcl-Form-Hide <CONTROL> {isHide [als Boolean]})
Rückgabewert Boolean (T, wenn erfolgreich, ansonsten NIL)
Anwendbar auf Alle Dialoge

Diese Methode versteckt einen Dialog oder zeigt ihn wieder an. Wird dem optionalen Argument isHide der Wert T mitgegeben, die der Dialog ausgeblendet, ansonsten wird er wieder eingeblendet. Bitte beachten Sie, dass das AutoCAD-Fenster deaktiviert bleibt, wenn Sie einen modalen Dialog ausblenden. Dieser Zustand sperrt AutoCAD.
