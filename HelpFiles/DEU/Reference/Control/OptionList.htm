Die Optionsliste ist eine dynamische Liste von Texten, von denen der Anwender eines aktivieren kann. Dabei werden alle anderen Optionen automatisch deaktiviert. Im Vergleich zum Optionsfeld kann die Anzahl der Optionen zur Laufzeit verändert werden.
