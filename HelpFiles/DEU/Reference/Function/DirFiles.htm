Funktionsname dcl-DirFiles
Syntax   (dcl-DirFiles strFolderPath [als String] {strFilter [als String]})
Rückgabewert Liste von Strings

Diese Funktion gibt eine Liste von Dateien im Verzeichnis strFolderPath zurück, die die Filterbedingungen in strFilter erfüllen. Wird kein Filter angegeben, werden alle Dateinamen zurückgegeben.
