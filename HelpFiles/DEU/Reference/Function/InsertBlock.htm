Funktionsname dcl-InsertBlock
Syntax   (dcl-InsertBlock strDwgFilePath [als String] {ptInsertionPoint [als 3DPoint]} {reaRotation [als Float]} {strBlockName [als String]})
Rückgabewert Boolean (T wenn erfolgreich; ansonsten NIL)

Diese Funktion fügt den Modellbereich einer Zeichnungsdatei (strDwgFilePath) in den aktuellen Bereich der aktuellen Zeichnungsdatei. Wird kein Einfügepunkt (ptInsertionPoint) angegeben, wird der Block am Koordinatenursprung eingefügt. Wird keine Drehung (reaRotation) angegeben, wird der Block mit dem Drehwinkel 0° eingefügt. Wird kein alternativer Blockname (strBlockName) angegeben, wird der Dateiname als Blockname in der aktuellen Zeichnungsdatei verwendet.

Diese Funktion dann sinnvoll einzusetzen, wenn der INSERT-Befehl nicht ausgeführt werden kann (z.B. zur Laufzeit eines OpenDCL-Ereignisses).
