Funktionsname dcl-XTwipsToPixels
Syntax   (dcl-XTwipsToPixels intTwips [als Long])
Rückgabewert Long

Diese Funktion konvertiert die Größe eines "twips" in Pixel. Ein Twip ist eine angloamerikanische Längeneinheit und bezeichnet die Teilung „TWentieth of an Inch Point“, also 1/20 Punkt = 1/1440 Zoll, wobei ein Punkt 1/72 eines Zolls sind, so dass die Konvertierung abhängig von der Bildschirmauflösung ist.
