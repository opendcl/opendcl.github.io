Funktionsname dcl-AttachXref
Syntax   (dcl-AttachXref strDwgFilePath [als String] {strBlockName [als String]} {ptInsertionPoint [als 3DPoint]} {reaRotation [als Float]})
Rückgabewert Boolean (T wenn erfolgreich; ansonsten NIL)

Diese Funktion ordnet die Datei strDwgFilePath als externe Referenz der aktuellen Zeichnungsdatei zu. ist der Einfügepunkt ptInsertionPoint nicht belegt, wird die externe Referenz am Koordinatenursprung eingefügt. Ist die Drehung reaRotation nicht definiert, wird die externe Referenz mit der Drehung 0° eingefügt. Wird kein alternativer Blockname für strBlockName übergeben, wird der zeichnungsinterne Name des XRef-Blocks der Name der XRef-Datei sein.

Diese Funktion dann sinnvoll einzusetzen, wenn der XREF-Befehl nicht ausgeführt werden kann (z.B. zur Laufzeit eines OpenDCL-Ereignisses).
