Funktionsname dcl-SelectFiles
Syntax   (dcl-SelectFiles {lstFileTypes [als Liste von Strings]} {strTitle [als String]} {strDefaultDirectory [als String]})
Rückgabewert String

Diese Funktion fordert den Anwender auf, eine oder mehrere Dateien zu wählen. Das optionale Argument lstFileTypes kann eine Liste der erlaubten Dateitypen sein. Dabei ist das Format "DateitypBezeichnung|DateiFilter" für jedes Format einzuhalten. Dabei ist DateitypBezeichnung die Bezeichnung in der Auswahlliste der Dateitypen (z.B. "Zeichnungsdatei") und DateiFilter ist ein Platzhalter, wonach Dateien ausgefiltert werden (z.B. "*.dwg"). Das optionale Argument strTitle ersetzt den Text in der Titelzeile des Dateiwahlfensters mit einem nutzerdefinierten Text. Wird das Argument strDefaultDirectory angegeben, wird das Dateiwahlfenster in diesem Verzeichnis geöffnet.

Die Funktion gibt eine Liste der gewählten Dateien einschließlich des kompletten Pfads zurück oder NIL, wenn keine Dateien markiert wurden oder der Dialog abgebrochen wurde.
