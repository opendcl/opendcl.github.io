Lokaler Name Use Visual Style
API Name UseVisualStyle
Datentyp Boolean
Gültigkeit  
Get Funktion   (dcl-Control-GetUseVisualStyle <CONTROL>)
Set Funktion   (dcl-Control-SetUseVisualStyle <CONTROL> isNewValue [als Boolean])
Anwendbar auf Winkel, Kontrollkästchen, Kombinationsfeld, Rahmen, Kombinationsfeld mit Bildern, Optionsfeld, Verlaufsbalken, Scrollleiste, Reglerschaltflächen, Schieberegler, Karteikarten, Textschaltfläche

Diese Eigenschaft bestimmt, ob die anwenderseits aktuell eingestellten Windows-spezifischen visuellen Stile (auch "Themen" genannt) verwendet werden sollen. Bitte beachten Sie, dass visuelle Stile die Einstellungen für die Hintergrund- und Vordergrundfarbe der Steuerelemente überschreiben können.
