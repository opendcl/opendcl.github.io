Lokaler Name Dockable Sides
API Name DockableSides
Datentyp Integer
Gültigkeit Gültige Werte sind Ganzzahlen zwischen 0 und 5.
Get Funktion   (dcl-Control-GetDockableSides <CONTROL>)
Set Funktion   (dcl-Control-SetDockableSides <CONTROL> intNewValue [als Integer])
Anwendbar auf Andockbares Fenster , Palette

Diese Eigenschaft bestimmt, an welchen Seiten andockbare Dialoge am AutoCAD-Fenster andocken dürfen.

Wert Beschreibung
0 links und rechts
1 nur oben
2 nur unten
3 oben und unten
4 links, rechts, oben
5 an allen Seiten
