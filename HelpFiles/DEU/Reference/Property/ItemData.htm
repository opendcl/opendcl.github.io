Lokaler Name Item Data
API Name ItemData
Datentyp Liste von Longs in der Form (intItem0Data intItem1Data ...)
Gültigkeit  
Get Funktion   (dcl-Control-GetItemData <CONTROL>)
Set Funktion   (dcl-Control-SetItemData <CONTROL> lstNewValue [als Liste von Longs])
Anwendbar auf Kombinationsfeld, Kombinationsfeld mit Bildern, Listenfeld

Diese Eigenschaft beinhaltet von Werten, die den Listenelementen als zusätzliche Ganzzahlwerte zugewiesen werden können (z.B. ID's als Schlüsselwerte aus Datenbanktabellen). Einige Stile von Kombinationsfeldern diese Art zusätzlicher Daten als Metadaten. So enthält jeder Eintrag der AutoCAD-Farbauswahl den entsprechenden ACI-Farbindex.
