Get Funktion   (dcl-Control-GetDisableNoScroll <CONTROL>)
Set Funktion   (dcl-Control-SetDisableNoScroll <CONTROL> isNewValue [als Boolean])
Anwendbar auf DWG-Liste, Listenfeld

Diese Eigenschaft bestimmt, ob Scrolleisten immer angezeigt werden sollen, selbst wenn nicht genügend Einträge vorhanden sind, so dass gescrollt werden könnte. In diesem Fall werden die Scrollleisten deaktiviert dargestellt. Hat diese Eigenschaft den Wert NIL, werden keine Scrollleisten angezeigt, wenn nicht genügend Einträge vorhanden sind. Beachten Sie bitte, dass eine Änderung der Eigenschaft keine Auswirkungen mehr hat, nachdem es erstellt wurde. Um das Ergebnis einer Änderung dieser Eigenschaft im OpenDCL Studio zu sehen, schließen Sie den Dialog und öffnen ihn erneut.

Bitte beachten Sie, dass diese Eigenschaft nur Effekte erzielen kann, wenn Sie der Eigenschaft Vertical Scroll Bar den Wert T zugewiesen haben.
