Lokaler Name ToolTip Picture
API Name ToolTipPicture
Datentyp PictureID
Gültigkeit -1 oder eine gültige PictureID
Get Funktion   (dcl-Control-GetToolTipPicture <CONTROL>)
Set Funktion   (dcl-Control-SetToolTipPicture <CONTROL> NewValue [als PictureID])
Anwendbar auf ActiveX Steuerelement, Winkel, Blockliste, Block, Kalender, Kontrollkästchen, Kombinationsfeld, DWG-Liste, DWG-Vorschau, Bildschaltfläche, Datenblatt, Schraffur, Hyperlink, Kombinationsfeld mit Bildern, Bezeichnung, Listenfeld, Erweiterte Liste, Optionsfeld, Bild, Verlaufsbalken, Dia, Textfeld, Textschaltfläche, Baumstruktur

Diese Eigenschaft bestimmt das Bild bzw. Symbol links neben dem Titel bzw. Haupttext des ToolTips, wenn der ToolTip angezeigt wird.
