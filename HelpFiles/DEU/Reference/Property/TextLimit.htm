Text Limit
Anmerkung zur deutschen Übersetzung melden
Deutschsprachiges Forum
Lokaler Name Text Limit
API Name TextLimit
Datentyp Long
Gültigkeit Must be greater than or equal to 0
Get Funktion   (dcl-Control-GetTextLimit <CONTROL>)
Set Funktion   (dcl-Control-SetTextLimit <CONTROL> intNewValue [als Long])
Anwendbar auf Kombinationsfeld, Kombinationsfeld mit Bildern, Textfeld

Diese Eigenschaft bestimmt die maximale Anzahl an Zeichen, die in dem Eingabefeld zulässig sind. Ist der Wert dieser Eigenschaft 0, wird die maximale Anzahl nur durch das Betriebssystem vorgegeben (typischerweise mehr als 32 kByte). Bitte beachten Sie, dass bereits enthaltener Text im Textfeld nicht geändert wird, wenn Sie nachträglich diese Eigenschaft ändern. Ändern Sie den Text, wenn Sie einen bestimmten Eintrag aus der Liste auswählen.
