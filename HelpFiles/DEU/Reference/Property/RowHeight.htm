Row Height

Lokaler Name Row Height
API Name RowHeight
Datentyp Long
Gültigkeit Must be greater than or equal to -1
Get Funktion   (dcl-Control-GetRowHeight <CONTROL>)
Set Funktion   (dcl-Control-SetRowHeight <CONTROL> intNewValue [als Long])
Anwendbar auf DWG-Liste, Datenblatt, Optionsliste

Diese Eigenschaft bestimmt die Höhe einer Zeile in der Liste. Der Wert 0 wendet die Standardhöhe an, der Wert -1 bewirkt, dass die Zeilenhöhe vom Inhalt der Zellen abhängig ist. Dieses Verhalten gilt nicht für das Datenblatt-Steuerelement, denn es unterstützt keine ungleichen Zeilenhöhen. Der Wert -1 bewirkt in diesem Fall die Änderung der Zeilenhöhe auf den Standardwert.
