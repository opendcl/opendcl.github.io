Lokaler Name Column Styles
API Name ColumnStyleList
Datentyp Liste von Integers in der Form (intColumn0Style intColumn1Style ...)
Gültigkeit Gültige Werte sind Ganzzahlen im Bereich zwischen 0 und 43
Get Funktion   (dcl-Control-GetColumnStyleList <CONTROL>)
Set Funktion   (dcl-Control-SetColumnStyleList <CONTROL> lstNewValue [als Liste von Integers])
Anwendbar auf Datenblatt

Beinhaltet die Liste der voreingestellten Zellenstile jeder Spalte. Die Zellen jeder Spalte verwenden den Vorgabestil, wenn dieser nicht für entsprechende Zellen individuell überschrieben werden.

Die Systemvariable DIMZIN beeinflusst die Darstellung von Realzahlen im Eingabefeld (Stil 9 und 39). Ist das Bit 3 gesetzt (Wert 4), werden führende Nullen unterdrückt. Die Systemvariable DIMZIN wird beim Wechsel des aktiven Bemaßungsstils geändert.
