Eine Instanz eines OpenDCL-Steuerelements kann mit Hilfe des Objektzeigers oder einer Reihe von Zeichenketten aus Projektname, Dialogname und Name des Steuerelements für das entsprechende Steuerelement identifiziert werden. Die Dokumentation von OpenDCL nutzt als Konvention die Bezeichnung <CONTROL>, wenn ein Steuerelement als Argument übergeben werden soll. In diesem Fall kann entwerder der Objektzeiger verwendet werden, mit dem das AutoLISP-Symbol belegt wurde, dessen Name in der Eigenschaft VarName eingetragen ist oder aber Projektname, Dialogname und Name des Steuerelements für das entsprechende Steuerelement Anwendung finden, wobei alle 3 als Strings übergeben werden.
