Image List

Image Lists sind Windows-Systemobjekte (Collections), die dazu dienen eine geordneten Liste von Bildern, Symbolen (Icons) zu speichern. Sie werden in Windows als Speicheradressen vorgehalten. OpenDCL stellt die Elemente der Bildliste als Objektzeiger zur Verfügung, wenn sie AutoLISP zurückgegeben werden. Diese Werte sind keine wirklichen Entitynames, wie man sie von AutoCAD kennt (die AutoLISP-Funktion (entget) kann hier also nicht angewandt werden). Sie können jedoch OpenDCL-Funktionen übergeben werden, die eine Image List erwarten und OpenDCL interpretiert sie korrekterweise als Speicheradresse einer Image List. Zur Organisation der Bilder und Symbole in den Image Lists stehen weitere Methoden zur Verfügung.
