Funktionsname c:<CONTROL-NAME>#OnCancelClose
Syntax   (defun c:<CONTROL-NAME>#OnCancelClose (intIsESC [als Integer] /)
)
Rückgabewert Boolean
Ausgelöst durch Dateiwahldialog, Modaler Dialog, Nichtmodaler Dialog

Dieses Ereignis fängt ab, ob der Dialog wirklich geschlossen werden soll (ausgelöst durch ESC-Taste oder Eingabetaste in Eingabefeldern). Der Rückgabewert dieser Funktion sollte T (/= intIsESC 1) sein, wenn der Dialog nicht geschlossen werden soll, NIL wenn doch.
