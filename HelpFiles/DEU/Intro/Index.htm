OpenDCL ist eine open source Entwicklungsumgebung für AutoLISP-Programmierer, die das limitierte DCL (Dialog Control Language) von AutoCAD ersetzen und auf einen umfangreichen Fundus von modernen Windows Steuerelementen zurückgreifen wollen. AutoCAD Applikationsentwickler und -Anwender können im OpenDCL Studio interaktiv neue moderne und professionelle Benutzerinterfaces designen und über ihren AutoLISP code auf die Steuerelemente und Dialoge zugreifen. Die OpenDCL Laufzeitumgebung wird mit der Applikation ausgeliefert und bietet innerhalb von AutoCAD die notwendige Umgebung für die Endanwender der OpenDCL-Applikation.
